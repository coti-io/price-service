# Price service

github-url: https://github.com/cotitech-io/price-service/issues

## Description
The price service is a monitor for price of coti & eth.  
in addition the service has an api to get price by given date from multiple sources.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
